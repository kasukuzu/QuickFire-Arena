# QuickFire Arena

React, TypeScript, React Three Fiber, and a WebSocket server prototype for a local 3D FPS deathmatch.

## Requirements

- Node.js 24 or newer
- npm

## Setup

```bash
npm install
npm run dev
```

Client: http://localhost:5173/

Server: ws://localhost:2567

Open two browser tabs, create a room in the first tab, join the room code from the second tab, then start the match from the host lobby.

## Checks

```bash
npm run typecheck
npm run build
```
